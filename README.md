# DeeVid Codex Skill

Install interactively:

```bash
npx skills add deevid-dev/deevid-skills
```

Install this skill directly:

```bash
npx skills add deevid-dev/deevid-skills --skill deevid-skill
```

This installs the full `deevid-skill` folder, including `SKILL.md` and the bundled CLI.

Set your API key:

```bash
export DEEVID_API_KEY="your_deevid_openapi_key"
```

Optional standalone CLI only:

```bash
npm install -g @deevid-ai/deevid-cli
```
